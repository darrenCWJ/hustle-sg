Sample Certificates

Use these as test certs for the cert parser demo, or reference them when adding your own real credentials. Open any cert, copy its text, and paste into the Cert Parser on the demo page to see Claude extract the details.

HOW TO USE

1
Click any cert below → opens the certificate HTML page.
2
Select all text on the page (Ctrl+A / Cmd+A) and copy it.
3
Go to your Profile → Edit → Certifications, paste the text into the cert parser, and let Claude extract issuer, title, and skills automatically.
4
For your real certificates — paste the text from your actual PDF or add the URL directly in the URL field. WSQ/polytechnic/university certs will be marked Verified.
